# Compiler Project

# This program is a complier for a C like programming languge that outputs MIPS code

# Features: 
* Int and Boolean types
* +, -, *, /, %
* ^ (exponentiation)
* <, <=, ==, >=, >, !=
* !, &&, ||
* Read ints from users
* Print lines, spaces, or a list of variables(ints)
* While loops
* For loops
* Single dimension arrays
* Boolean expressions
